# Xyra Termux WebView App

A minimal Android WebView app that loads https://xyra-termux.vercel.app/

## Quick Start in AIDE
1. Open `AIDE/XyraTermux` folder in AIDE
2. Build → Build APK
3. Install on your Android device

## Features
- ✅ WebView displays remote web content
- ✅ JavaScript enabled
- ✅ Back button navigation support
- ✅ Minimal dependencies

## Architecture
- **MainActivity.java** - WebView activity
- **AndroidManifest.xml** - App configuration
- **build.gradle** - Gradle build setup

No complex resources - just pure WebView functionality.
